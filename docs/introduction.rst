Introduction
============

This :py:mod:`ledsign` module allows the direct control and programming of the LED Sign series of devices using Python 3, or through a dedicated CLI interface.

Installation
------------

The :code:`ledsign` is available for easy download via PyPI. It can be installed with :code:`pip`:

.. code-block:: bash

	$ pip install ledsign

Alternatively, development versions can be downloaded directly from the `GitHub repository <https://github.com/krzem5/ledsign>`_.
